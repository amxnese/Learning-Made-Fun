6. What is recursion in Python?

7. Consider the following recursive function. What is its purpose?

def func(n):
  if n == 1:
    return 1
  else:
    return n * func(n - 1)
print(func(5))

8. What will the following recursive function output?

def fibonacci(n):
  if n <= 1:
    return n
  else:
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(7))
