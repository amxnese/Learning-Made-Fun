What is recursion in Python?

Consider the following recursive function. What is its purpose?

            
              def factorial(n):
                  if n == 1:
                      return 1
                  else:
                      return n * factorial(n - 1)

              print(factorial(5))

What will the following recursive function output?

            
              def fibonacci(n):
                  if n <= 1:
                      return n
                  else:
                      return fibonacci(n-1) + fibonacci(n-2)

              print(fibonacci(7))
