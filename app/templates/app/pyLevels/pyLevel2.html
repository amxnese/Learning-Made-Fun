Python's Built-in Functions

What are built-in functions

Built-in functions are pre-defined functions provided by a programming language's standard library, allowing you to perform common tasks without having to implement the functionality yourself. These functions are "built into" the language, meaning they are always available and can be used without any additional imports or definitions.

Why we use built-in functions

  • Efficiency: These functions are generally optimized for performance and are faster than equivalent functions you might write on your own.
  • Convenience: They simplify common tasks like mathematical operations, input/output processing, and data manipulation.
  • Reliability: Built-in functions are thoroughly tested as part of the language's development, ensuring that they work correctly in all expected scenarios.

Most used functions

Function Description
print() Displays the specified message to the screen, or other standard output device.
len() Returns the number of items in an object.
type() Returns the type of the specified object.
int(), str(), float() Convert objects to integer, string, or float types respectively.
input() Allows user input.
range() Generates a sequence of numbers, often used in for loops.
append() Adds an item to the end of a list.
sorted() Returns a sorted list from the provided iterable.
sum() Sums the items of an iterator.
max(), min() Return the largest or smallest item in an iterable.
map() Applies a given function to each item of an iterable and returns a list of the results.
filter() Constructs an iterator from elements of an iterable for which a function returns true.
zip() Returns an iterator of tuples, where the i-th tuple contains the i-th element from each of the argument sequences or iterables.
enumerate() Adds a counter to an iterable and returns it in a form of enumerating object.

Commmon Examples

  • 1. 'print()' Function

    The 'print()' function is used to output text or variables to the console. It's one of the most frequently used functions in Python, especially for debugging purposes. 

    print("Hello, world!")

  • 2. 'len()' Function

    The 'len()' function returns the number of items in a container. This function is useful when you need to know the size of a list, string, or any other collection. 

    # Example of using len() to get the length of a list
my_list = [10, 20, 30, 40, 50]
print("The length of the list is:", len(my_list))

  • 3. 'type()' Function

    The 'type()' function returns the type of the specified object. This can be helpful for debugging or when you're working with data and need to know what kind of variable you're dealing with. 

    # Example of using type() to get the type of a variable
my_variable = 3.14
print("The type of my_variable is:", type(my_variable))

  • 4. 'int()' Function

    The int() function is used to convert a compatible value into an integer. This is very useful when you need to ensure that numerical data is in integer form for calculations or indexing. 

    # Example of using int() to convert a string to an integer
number_string = "123"
converted_number = int(number_string)
print("Converted number:", converted_number)

  • 5. input() Function

    The input() function allows you to take user input from the console. It is useful for interactive programs where you need input from the user. 

    # Example of using input() to get user input
user_name = input("Enter your name: ")
print("Hello, " + user_name + "!")

  • 6. The 'sum()' Function

    The 'sum()' function in Python is used to add up all the numbers in an iterable, such as a list or tuple, and return the total. Here's a simple example that demonstrates how to use the sum() function: 

    # Example of using sum() to calculate the total of numbers in a list
numbers = [5, 10, 15, 20, 25]
total = sum(numbers)
print("The total sum of the numbers is:", total)

    In this example, sum() takes the list numbers as its argument and returns the sum of its elements, which is then printed out.
