Loops in Python

What are Loops

Loops are a fundamental programming concept that allows a block of code to be executed repeatedly based on a condition.

Python has two primitive loop commands:
  • while loops
  • for loops

for loop:

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string). This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages. With the for loop we can execute a set of statements, once for each item in a list, tuple, set etc. It is used when the number of iterations is known.

while loop:

Repeats as long as a certain boolean condition is met. It is used when the number of iterations is not known before the loop starts.

Why Use Loops in Programming?

Loops are a fundamental component of programming that allow us to perform repetitive tasks efficiently. Below are some of the primary reasons we use loops:

  • Automation: Loops automate repetitive tasks, reducing the need to write the same code multiple times.
  • Efficiency: Using loops, we can traverse through data structures like lists, arrays, or strings, and manipulate large amounts of data with minimal code.
  • Control: Loops give programmers control over how many times certain operations are executed, based on dynamic conditions.
  • Simplicity: Loops simplify complex problems into manageable tasks by breaking them down into repeated subtasks.

When to Use Loops in Programming

Understanding when to utilize loops in your code is crucial for writing efficient and readable programs. Here are some common scenarios where loops are particularly useful:

  • Processing Collections: Use loops to iterate through elements in a collection such as a list, tuple, or dictionary to perform operations on each element.
  • Repeating Actions: When you need to repeat a specific action multiple times, loops make your code more compact and maintainable.
  • Conditional Execution: Loops can run a block of code until a certain condition changes (like receiving a specific user input).
  • Generating Sequences: Loops are ideal for generating sequences of numbers, processing series, or creating repeated patterns.
  • Automating Tasks: Any task that requires automation, such as sending emails to a list of users, can be efficiently managed with loops.

Keywords Related to Loops in Python

In Python, loops are used to execute a block of code repeatedly under certain conditions. Here are some key terms and keywords associated with looping in Python:

  • for - Used to iterate over a sequence (like a list, tuple, dictionary, set, or string).
  • while - Repeats a block of code as long as a condition is true.
  • break - Exits the loop immediately, regardless of the iteration condition.
  • continue - Skips the current iteration of the loop and moves to the next iteration.
  • else - Optionally used after a loop to execute a block of code once when the loop condition is no longer true.
  • range() - Often used with for loops to generate a sequence of numbers.
  • enumerate() - Used to loop through a sequence but it keeps track of both the index and the value.
  • iter() - Used to create an iterator object, and can be used with a loop to iterate through the elements.
  • next() - Used to get the next item from an iterator.

Python Loop Examples

Understanding loops in Python can be greatly enhanced with practical examples. Here are some common examples of using both for and while loops in Python:

Example 1: Using a for loop to iterate over a list

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
  print("I like", fruit)
# output:
'''
I like apple
I like banana
I like cherry
'''

Example 2: Using a while loop to execute until a condition is met 

count = 1
while count <= 5:
  print("Count is:", count)
  count += 1

this loop will stay on until count is greater than or equal to five

Example 3: Using a for loop with the range function

for number in range(1, 11):
  print("Number is", number)

this loop will iterate on the print function 10 times and will print numbers from 1 to 10. Note that the first initial value is included white the final value is excluded

Example 4: Nested loops to create a pattern

for i in range(1, 4):  # Outer loop
  for j in range(1, 4):  # Inner loop
    print(f"i = {i}, j = {j}")

These examples showcase different ways to use loops in Python, from iterating through collections to using nested loops for complex patterns.
