Conditional Statements in Python

What are Conditional Statements

Conditional statements in Python are used to execute different blocks of code based on certain conditions. These statements are essential for decision making in your code. The primary conditional statements in Python are:

  • if: This is used to test a condition and execute a block only if the condition is true.
  • elif: Short for "else if", it allows you to test multiple conditions after an if statement. It's executed if its condition is true and all preceding conditions in the chain are false.
  • else: This catches anything which isn't caught by the preceding conditions.

Conditional statements are a fundamental part of programming, allowing you to control the flow of execution based on conditions.

When To Use Conditional Statements

Conditional statements in programming, including Python, are used in various scenarios where you need to make decisions based on specific conditions. These are some common use cases:

  • Control Flow: To direct the flow of execution depending on whether certain conditions are true or false. For example, performing different actions based on user input.
  • Data Filtering: To execute code only for items that meet certain criteria, such as filtering data in a list based on some attributes.
  • Error Handling: To check for potential errors or unexpected values before performing operations. For example, checking if a divisor is zero before performing a division.
  • Feature Toggle: Enabling or disabling features in software based on configuration options or environment variables.
  • User Authentication: To determine the level of access or permissions a user has, based on their credentials or roles.
  • Validation: To ensure that input data meets certain criteria before it's processed further, such as checking if a user's input is within a valid range.

Examples on Python's Conditional Statements

Here are examples of how to use if, elif, and else statements

Simple if, elif, and else Example

This example demonstrates basic conditional logic:

            
              x = 20
              if x < 10:
                print("Less than 10")
              elif x < 20:
                print("Less than 20 but not less than 10")
              else:
                print("20 or more")

Nested Conditional Example

Nested conditionals are useful for more complex decision structures. Here is how you can use a nested if inside another if: 

            
              age = 25
              registered = True
              if registered:
                if age < 18:
                  print("Registered but under 18")
                else:
                  print("Registered and 18 or older")
              else:
                print("Not registered")

Conditional statements are crucial for controlling the flow of execution in a program, allowing for different actions based on different conditions.
