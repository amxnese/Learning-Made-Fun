JavaScript's Built-in Data Structures

What are built-in Data Structures

In JavaScript, built-in data structures are integral types that facilitate the storage, organization, and management of data within programs. These data structures come with a variety of operations for efficiently handling data, such as adding, removing, searching, and accessing elements. They are native to the JavaScript language, meaning they can be used directly without the need for importing external libraries. Each data structure has unique properties and methods that are optimized for specific use cases, providing developers with the tools necessary to handle data in diverse and effective ways.

Why we use built-in Data Structures

There are several reasons why built-in data structures are used in JavaScript:

  • Efficiency: Built-in data structures like arrays and objects are optimized for performance, making operations such as access, insertion, and deletion fast and efficient.
  • Convenience: These data structures come with built-in methods and properties that simplify common tasks, reducing the need for manual implementation and potential errors.
  • Standardization: Using built-in data structures promotes consistency and interoperability across different JavaScript environments and libraries, enhancing code maintainability and collaboration.
  • Compatibility: Since these data structures are part of the JavaScript language specification, they are widely supported across browsers and platforms, ensuring compatibility with a broad range of development environments.

Available Data Structures in JavaScript

1. Array:

  • An Array is a dynamic list-like object that can hold elements of any data type.
  • We Use Array when you need to store a collection of elements and access them by index.
  • Example:
            
              let arr = ["Apple", "Banana"];
              arr.push("Orange"); // Adding element to the end
              let fruit = arr[0]; // Accessing element by index

2. Set:

  • Set is a collection of unique values.
  • Use Set when you need to store unique elements and perform operations like checking for existence or removing duplicates.
  • Example:
            
              let set = new Set();
              set.add("Apple");
              set.add("Banana");
              let containsApple = set.has("Apple"); // Checking if set contains an element

3. Map:

  • Map is a collection of key-value pairs where each key is unique.
  • Use Map when you need to associate values with keys and perform operations like lookup, insertion, and deletion based on keys.
  • Example:
            
              let map = new Map();
              map.set("apple", 10);
              map.set("banana", 5);
              let quantity = map.get("apple"); // Retrieving value by key

4. Object:

  • Object is a collection of key-value pairs where keys are strings.
  • Use Object when you need to create a custom data structure with properties and methods.
  • Example:
            
              let obj = {
                name: "John",
                age: 30
              };
              let name = obj.name; // Accessing property by key

Some Built-in Functions Related To Built-in Datastrures

Arrays

  • length: Returns the number of elements in an array.
  • push(): Adds one or more elements to the end of an array and returns the new length of the array.
  • pop(): Removes the last element from an array and returns that element.
  • sort(): Sorts the elements of an array in place and returns the sorted array.

Examples:

            
              let myArray = [3, 1, 4, 1, 5, 9, 2, 6, 5];
              console.log(myArray.length);  // Output: 9

              myArray.push(7);
              console.log(myArray);  // Output: [3, 1, 4, 1, 5, 9, 2, 6, 5, 7]

              let poppedItem = myArray.pop();
              console.log(poppedItem);  // Output: 7
              console.log(myArray);  // Output: [3, 1, 4, 1, 5, 9, 2, 6, 5]

              myArray.sort();
              console.log(myArray);  // Output: [1, 1, 2, 3, 5, 5, 6, 9]

Objects

  • Object.keys(): Returns an array of a given object's own enumerable property names.
  • Object.values(): Returns an array of a given object's own enumerable property values.
  • Object.entries(): Returns an array of a given object's own enumerable property [key, value] pairs.
  • Object.assign(): Copies the values of all enumerable own properties from one or more source objects to a target object.

Examples:

            
              let myObject = {a: 1, b: 2, c: 3};

              console.log(Object.keys(myObject));  // Output: ["a", "b", "c"]
              console.log(Object.values(myObject));  // Output: [1, 2, 3]
              console.log(Object.entries(myObject));  // Output: [["a", 1], ["b", 2], ["c", 3]]

              let copiedObject = Object.assign({}, myObject);
              console.log(copiedObject);  // Output: {a: 1, b: 2, c: 3}

Sets

  • add(): Adds a new element to the set.
  • delete(): Removes a specified element from the set.
  • has(): Returns a boolean indicating whether an element exists in the set or not.
  • forEach(): Executes a provided function once for each set element.

Examples:

            
              let mySet = new Set([1, 2, 3]);

              mySet.add(4);
              console.log(mySet);  // Output: Set { 1, 2, 3, 4 }

              mySet.delete(2);
              console.log(mySet);  // Output: Set { 1, 3, 4 }

              console.log(mySet.has(3));  // Output: true
              console.log(mySet.has(2));  // Output: false

              mySet.forEach(item => console.log(item * 2));
              // Output:
              // 2
              // 6
              // 8
