Getting Familiar With JavaScript's Syntax

JavaScript Syntax

In JavaScript, syntax is the set of rules that defines how to write JavaScript code. JavaScript syntax borrows heavily from C and Java, but it's also influenced by Awk, Perl, and Python. Here's an overview of the basic syntax

Variables

Variables can be declared using var, let, or const.

let name = "John";
const age = 30;
var height = 180;

Data Types

JavaScript is a loosely typed language, which means you don't have to declare the data type of a variable explicitly. The main data types are:

  • Number
  • String
  • Boolean
  • Object
  • Function
  • Array
  • Undefined
  • Null

Examples on some data types

  • Number: In JavaScript, a "Number" is a primitive data type used to represent numerical values, including integers and floating-point numbers. It is used for mathematical operations and can store both positive and negative values, as well as decimal fractions. 

    let num = 10;

  • String: In JavaScript, a "String" is a sequence of characters enclosed within single quotes ('') or double quotes (""). It is a primitive data type used to represent text data. Strings can contain letters, numbers, symbols, and whitespace. They are widely used for storing and manipulating textual information in JavaScript programs 

    let message = "Hello";

  • Boolean: A "Boolean" in JavaScript is a primitive data type that represents a logical value. It can have one of two possible values: true or false. Booleans are commonly used for making decisions in code, controlling the flow of execution, and performing conditional operations. 

    let status = true;

Operators

In JavaScript, operators are used to perform operations on variables and values. JavaScript supports several types of operators, each serving different purposes:

Arithmetic Operators

These operators are used for performing mathematical calculations:

  • '+' (Addition): Adds two operands.
  • '-' (Subtraction): Subtracts one operand from another.
  • '*' (Multiplication): Multiplies two operands.
  • '/' (Division): Divides the first operand by the second.
  • '%' (Modulus): Returns the remainder of a division.
  • '++' (Increment): Increases an integer value by one.
  • '--' (Decrement):Decreases an integer value by one

Assignment Operators

These operators are used to assign values to JavaScript variables:

  • '=' (Assignment): Assigns a value to a variable.
  • '+=' (Add and assign): Adds a value to a variable and assigns the result to that variable.
  • '-=' (Subtract and assign): Subtracts a value from a variable and assigns the result to that variable.
  • '*=' (Multiply and assign): Multiplies a variable by a value and assigns the result to that variable.
  • '/=' (Divide and assign): Divides a variable by a value and assigns the result to the variable.
  • '%=' (Modulus and assign): Takes modulus using two operands and assigns the result to the variable.

String Operators

The '+' operator can also be used to concatenate strings:

'+' (Concatenation): Combines two string values to create a new string.
