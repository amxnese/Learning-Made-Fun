Conditional Statements in JavaScript

What Are Conditional Statements

Conditional statements in JavaScript are used to perform different actions based on different conditions. These statements allow your script to make decisions and execute certain code blocks when certain conditions are met. Here are the main types of conditional statements in JavaScript:

  • if statement: Executes a block of code if the specified condition is true. 
    if (condition) {
  // code to be executed if condition is true
}
  • if-else statement: Executes a block of code if the condition is true; otherwise, it executes another block of code. 
    • if (condition) {
  // code to be executed if condition is true
} else {
  // code to be executed if condition is false
}
  • else-if ladder: Provides a way to decide among several blocks of code to be executed. 
    if (condition1) {
  // code to be executed if condition1 is true
} else if (condition2) {
  // code to be executed if condition2 is true
} else {
  // code to be executed if both condition1 and condition2 are false
}
  • switch statement: Executes one code block from many choices. Unlike if statements that evaluate conditions, the switch statement selects execution paths based on an expression's value. 
    switch (expression) {
  case x:
    // code block
    break;
  case y:
    // code block
    break;
  default:
    // default code block
}

When to Use Conditional Statements in JavaScript

Conditional statements are fundamental to programming, allowing you to make decisions in your code based on conditions. Here are common scenarios where conditional statements are used in JavaScript:

  • Decision Making: Use conditional statements to execute different code based on various conditions. This is helpful in scenarios such as displaying different messages based on user input or deciding which action to take next.

  • Form Validations: Before submitting form data to a server, you might want to ensure that the data is correct. Conditional statements can check things like whether an email address is in the correct format, or whether all required fields are filled out.

  • Feature Access Control: Conditional statements can control access to certain parts of your application, such as enabling or disabling features based on user roles or subscription levels.

  • Handling Different Scenarios: In real-world applications, you often need to handle various scenarios that depend on external data. For example, displaying different content on a website based on the time of day, or showing different interfaces to users based on the device they are using.

  • Error Handling: Use conditional statements to check for potential errors in your code, such as checking if an object exists before accessing its properties to avoid runtime errors.

Keywords Related to Conditional Statements in JavaScript

Conditional statements are crucial in JavaScript for decision-making processes. Here are some important keywords and operators related to conditional statements:

  • if - Executes a block of code if a specified condition is true.
  • else - Executes a block of code if the condition in the if statement is false.
  • else if - Specifies a new condition to test, if the first condition is false.
  • switch - Allows a block of code to be executed among many alternatives. Uses case statements to compare the given value.
  • case - Specifies a block of code to run if it matches the value in the switch statement.
  • default - Specifies the code to run if there is no case match in a switch statement.
  • ?: (ternary operator) - Provides a shorthand way of writing an if-else statement. It takes three operands and evaluates to a value based on the first condition being true or false.
  • break - Terminates the current loop, switch, or label statement and transfers program control to the statement following the terminated statement.

Examples of Using Conditional Statements in JavaScript

Conditional statements control the flow of execution based on conditions. Below are examples of how these statements can be used:

  • Using if statement:

    A basic example to display a message based on the age. 

    // Check if someone is old enough to vote
var age = 20;
if (age >= 18) {
  console.log("You are eligible to vote.");
} else {
  console.log("You are not eligible to vote.");
}

  • Using else-if ladder:

    Multiple conditions to handle different ranges of grades. 

    // Determine the grade of a student based on their score
var score = 85;
if (score >= 90) {
  console.log("Grade A");
} else if (score >= 80) {
  console.log("Grade B");
} else if (score >= 70) {
  console.log("Grade C");
} else if (score >= 60) {
  console.log("Grade D");
} else {
  console.log("Grade F");
}

  • Using switch-case:

    Different cases for handling specific user commands. 

    // Respond to a user command
var command = 'start';
switch (command) {
  case 'start':
    console.log("Starting the engine...");
    break;
  case 'stop':
    console.log("Stopping the engine...");
    break;
  default:
    console.log("Command not recognized.");
}

  • Nested if statements:

    Checking multiple conditions within a condition. 

    // Check login status and user role
var loggedIn = true;
var userRole = 'admin';
if (loggedIn) {
  if (userRole === 'admin') {
    console.log("Access granted to admin panel.");
  } else {
    console.log("Access granted. Limited permissions.");
  }
} else {
  console.log("Please log in to continue.");
}
