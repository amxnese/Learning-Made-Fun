What are built-in functions

In JavaScript, a built-in function typically refers to the methods provided by Java's standard libraries, which are part of the Java API. These methods are readily available for use without the need for the programmer to define them. They serve a wide range of purposes, from mathematical calculations to handling strings, files, and data collections.

Why we use built-in functions

We use built-in functions in programming languages like Java for several important reasons:

  • Efficiency: Built-in functions are often optimized for performance by the language developers. They are implemented in a way that maximizes speed and minimizes resource usage, which can be hard to achieve with custom code.
  • Reliability: These functions have been thoroughly tested and are used widely, ensuring that they are reliable and free from common bugs. This reduces the risk of errors in your own code.
  • Simplicity: Built-in functions simplify the coding process by providing ready-to-use solutions for routine tasks, allowing developers to accomplish more with less code. This makes the code cleaner and easier to understand.
  • Portability: Using built-in functions ensures that your code can run on any platform that supports the Java runtime, without needing any modifications. This is because the functionality of these functions is defined as part of the Java standard.
  • Productivity: By reducing the need to write and test custom implementations of common functionalities, built-in functions allow developers to focus more on the specific logic and features unique to their applications.
  • Standardization: Built-in functions promote coding standards and best practices. When developers use these functions, it helps maintain a uniformity in coding styles across different projects, which is beneficial for code maintenance and team collaboration.

Most Used Built-in JavaScript Functions

Function Description
alert() Displays an alert dialog with the specified message and an OK button.
console.log() Outputs information to the web console.
document.getElementById() Returns the element that has the ID attribute with the specified value.
document.querySelector() Returns the first element within the document that matches the specified selector, or group of selectors.
document.querySelectorAll() Returns a static NodeList representing a list of the document's elements that match the specified group of selectors.
document.createElement() Creates a new element with the specified tag name.
addEventListener() Sets up a function that will be called whenever the specified event is delivered to the target.
setTimeout() Sets a timer which executes a function or specified piece of code once after the timer expires.
clearTimeout() Cancels a timeout previously established by calling setTimeout().
setInterval() Repeatedly calls a function or executes a code snippet, with a fixed time delay between each call.

Examples on some usefull functions

  • 1. 'alert()' Function

    The 'alert()' function is used to display a modal dialog with a specified message and an OK button to the user. It's a part of the window object in JavaScript and commonly used for alerting the user in a variety of scenarios, primarily for quick debugging or simple notifications. 

                  
                // Example of using alert() to show a greeting
                alert("Hello, world!")

  • 1. 'console.log()' Function

    The 'console.log()' function is used to output information to the web console, which is an essential tool for debugging JavaScript code. It can display strings, variables, objects, and more, making it incredibly useful for inspecting the internal state of an application or tracking the flow of execution. 

                  
                // Example of using console.log() to display a message and a variable
                var message = "The value is:";
                var value = 10;
                console.log(message, value);

    This example illustrates how console.log() can be used to log both text and variable values to the console, providing a practical way to track values and application state during development.

  • 1. 'document.getElementById()' Function

    The 'document.getElementById()' function is used to retrieve an element from the HTML document by its ID attribute. This function is crucial for manipulating HTML elements directly with JavaScript, allowing developers to update content, styles, and interact with specific parts of the web page dynamically. 

                  
                // Example of using document.getElementById() to change text content
                var myElement = document.getElementById("demo");
                myElement.textContent = "Hello, JavaScript!";

    This example demonstrates how to select an HTML element with the ID "demo" and change its text content to "Hello, JavaScript!" using JavaScript. This is a common approach for dynamically updating the content of web pages.

  • 1. 'document.querySelector()' Function

    The 'document.querySelector()' function is used to select the first element within the document that matches a specified CSS selector(s). It's incredibly versatile, allowing developers to access any element by leveraging CSS syntax, which makes it ideal for more complex querying needs beyond simple ID or class selectors. 

                  
                // Example of using document.querySelector() to find and style a button
                var button = document.querySelector("button.primary");
                button.style.backgroundColor = "blue";

    This example shows how to select the first button element with a class of "primary" and change its background color to blue. document.querySelector() is a powerful tool for accessing elements in a flexible and precise manner.

  • 1. 'document.createElement()' Function

    The 'document.createElement()' function is used to create a new element node in the DOM. This function is essential for dynamically adding new elements to the webpage. Developers use it extensively to build UI components on the fly, modify content, and interact with the document structure programmatically. 

                  
                // Example of using document.createElement() to create a new div and add it to the body
                var newDiv = document.createElement("div");
                newDiv.innerHTML = "Hello, dynamic world!";
                document.body.appendChild(newDiv);

    In this example, a new <div> element is created, its inner HTML is set to "Hello, dynamic world!", and then it is appended to the body of the document. This method is commonly used to dynamically add content to a web page.

  • 1. 'addEventListener()' Function

    The 'addEventListener()' method is used to attach an event handler to a specific element. This method provides a way to execute a function when a specified event occurs, such as a mouse click, a key press, or a page load. It is crucial for interacting with users and handling events in an efficient, non-obtrusive way. 

                  
                // Example of using addEventListener() to handle click events
                var btn = document.querySelector("button");
                btn.addEventListener("click", function() {
                  alert("Button was clicked!");
                });

    This example demonstrates how to attach a click event listener to a button. When the button is clicked, it triggers an alert that says "Button was clicked!". The addEventListener() method is versatile and can be used to handle all sorts of events across different elements.

  • 1. 'setTimeout()' Function

    The 'setTimeout()' function is used to execute a piece of code or a function after a specified delay (in milliseconds). It's part of the Window or Worker global scope, and it's typically used for delaying function execution, creating timeouts, or scheduling tasks to run after an interval. 

                  
                // Example of using setTimeout() to execute a function after a 3-second delay
                setTimeout(function() {
                  console.log("This message appears after 3 seconds");
                }, 3000);

    This example shows how to use setTimeout() to log a message to the console after a delay of 3000 milliseconds (or 3 seconds). The function allows for scheduling of both simple and complex tasks that should not execute immediately, providing flexibility in timing and execution flow control.

  • 1. 'setInterval()' Function

    The 'setInterval()' function is used to repeatedly execute a function or a block of code at fixed time intervals, specified in milliseconds. It is part of the Window or Worker global scope and is commonly used for tasks that require periodic updates, such as updating a countdown timer, animating elements in a loop, or polling a server. 

                  
                // Example of using setInterval() to increment and display a counter every second
                var count = 0;
                var intervalId = setInterval(function() {
                  count++;
                  console.log("Count: " + count);
                  if (count >= 5) {
                    clearInterval(intervalId); // Stop the interval after 5 iterations
                    console.log("Interval stopped.");
                  }
                }, 1000);

    This example demonstrates how to use setInterval() to create a simple counter that increments every second. The interval is stopped after reaching a count of 5 by calling clearInterval(), which is used to cancel the interval. This method is essential for any task that requires regular, repeated execution at set time intervals.
