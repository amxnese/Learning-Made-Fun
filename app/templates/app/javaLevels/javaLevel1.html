Getting Familiar With Java's Syntax

Java Syntax

Java is a widely-used, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. Here's a brief overview of its syntax:

Comments

In Java, comments are notes written in the source code by programmers to explain and clarify the code they write. These comments are not executed as part of the program and are mainly used for making the code easier to understand for humans. 

          
        // This is a single-line comment.
        int number = 10;// This comment follows a code statement.

Basic Structure

A Java program can be thought of as a collection of objects that communicate via invoking each other's methods. Here's how a simple Java program looks: 

            
        public class Main {
          public static void main(String[] args){
            System.out.println("Hello, world!");
          }
        }
  • Class Declaration:The public class Main line declares a class named Main.
  • Method Declaration:Inside the class, we can define methods. Here, public static void main(String[] args) is the main method, which is the entry point of any Java program.
  • Statements: Inside the method, we write our logic. System.out.println("Hello, world!"); is a statement that prints the string "Hello, world!" to the console.

Variables

In Java, a variable is a piece of memory that can contain data which may be modified during program execution. Variables are basic units of storage in a Java program. Each variable has a specific type, which determines the size and layout of the variable's memory, the range of values that can be stored within that memory, and the set of operations that can be applied to the variable.

Here are some key points about variables in Java:

  • Type: Java is a statically typed language, meaning that every variable must be declared with a type before it can be used. The type can be a primitive type like int, double, char, or boolean, or it can be an object type (including arrays and strings).
  • Declaration: Declaring a variable involves specifying the type and giving the variable a name. For example: 
                    
        int age;
        String name;
  • Initialization: You can initialize a variable at the time of declaration by assigning it a value: 
                    
        int age = 30;

Data Types

Java, a strongly-typed programming language, has a variety of data types that are categorized mainly into two groups: primitive data types and reference data types. Here's a brief overview of some:

Primitive Data Types:

Primitive data types are predefined by the language and named by a reserved keyword. They represent the most basic types of data Java can handle directly.

Numeric Types:

  • Integer Types
    • byte: 8-bit signed integer (-128 to 127)
    • short: 16-bit signed integer (-32,768 to 32,767)
    • int: 32-bit signed integer (-2^31 to 2^31-1)
    • long: 64-bit signed integer (-2^63 to 2^63-1) 
      •                 
        byte my_byte = 20;
        short my_short = 120;
        int my_int = 5500;
        long my_long = 12000;
  • Floating Point Types
    • float: 32-bit floating-point (single precision)
    • double: 64-bit floating-point (double precision) 
      •                 
        float my_float = 2.71;
        double my_double = 3.14;

Character Type

  • char: 16-bit Unicode character (0 to 65,535)
    •               
        char my_char = 'A';

Boolean Type

  • boolean: Represents true/false values
    •               
              boolean myBoolean = false;

Special Mention: String

  • String: Although String is technically a class, it has some characteristics that make it unique in Java, such as being immutable and having operators like + for concatenation. 
                    
        String  = "Hello world!";

Operators

In Java, operators are special symbols that perform specific operations on one, two, or three operands, and then return a result. Java supports several types of operators, which can be categorized into several groups:

Arithmetic Operators

These operators are used for performing basic mathematical operations:

  • '+' (Addition): Adds two values together.
  • '-' (Subtraction): Subtracts one value from another.
  • '*' (Multiplication): Multiplies two values.
  • '/' (Division): Divides one value by another.
  • '%' (Modulus): Returns the remainder of a division operation.

Assignment Operators

Used to assign values to variables.

  • '=' Assigns values from right side operands to left side operand.
  • '+=' Adds right operand to the left operand and assigns the result to the left operand.
  • '-=' Subtracts the right operand from the left operand and assigns the result to the left operand.
  • '*=' Multiplies right operand with the left operand and assigns the result to the left operand.
  • '/=' Divides left operand by the right operand and assigns the result to the left operand.
  • '%=' Takes modulus using two operands and assigns the result to the left operand.
  • '**/' Performs exponential (power) calculation on operators and assigns value to the left operand.
  • '//=' Performs floor division on operators and assigns value to the left operand.
