What are built-in functions

In Java, a built-in function typically refers to the methods provided by Java's standard libraries, which are part of the Java API. These methods are readily available for use without the need for the programmer to define them. They serve a wide range of purposes, from mathematical calculations to handling strings, files, and data collections.

What are built-in functions used for

We use built-in functions in programming languages like Java for several important reasons:

  • Efficiency: Built-in functions are often optimized for performance by the language developers. They are implemented in a way that maximizes speed and minimizes resource usage, which can be hard to achieve with custom code.
  • Reliability: These functions have been thoroughly tested and are used widely, ensuring that they are reliable and free from common bugs. This reduces the risk of errors in your own code.
  • Simplicity: Built-in functions simplify the coding process by providing ready-to-use solutions for routine tasks, allowing developers to accomplish more with less code. This makes the code cleaner and easier to understand.
  • Portability: Using built-in functions ensures that your code can run on any platform that supports the Java runtime, without needing any modifications. This is because the functionality of these functions is defined as part of the Java standard.
  • Productivity: By reducing the need to write and test custom implementations of common functionalities, built-in functions allow developers to focus more on the specific logic and features unique to their applications.
  • Standardization: Built-in functions promote coding standards and best practices. When developers use these functions, it helps maintain a uniformity in coding styles across different projects, which is beneficial for code maintenance and team collaboration.

Commonly used examples

  • 1. Math Functions

    • Math.abs()

      This Java code snippet declares an integer variable a with a value of -20. It then prints the absolute value of a using the Math.abs() method. The output will be "Absolute value of -20 is: 20", as Math.abs() converts any negative number to its positive equivalent. 

                          
                      int a = -20;
                      System.out.println("Absolute value of -20 is: " + Math.abs(a));

    • Math.sqrt()

      This Java code snippet declares a double variable b with a value of 9.0. It then prints the square root of b using the Math.sqrt() method. The output will be "Square root of 9 is: 3.0", as Math.sqrt() calculates the square root of the provided number. 

                          
                      double b = 9.0;
                      System.out.println("Square root of 9 is: " + Math.sqrt(b));

    • Math.pow()

      This Java code snippet declares two double variables base with a value of 2.0 and exponent with a value of 3.0. It then prints the result of raising base to the power of exponent using the Math.pow() method. The output will be "2 raised to the power of 3 is: 8.0", as Math.pow() calculates the power of the base raised to the given exponent. 

                          
                      double base = 2.0;
                      double exponent = 3.0;
                      System.out.println("2 raised to the power of 3 is: " + Math.pow(base, exponent));

  • 2. String Functions

    • String.length()

      This Java code snippet initializes a String variable str with the value "Hello, world!". It then prints the length of the string using the length() method of the String class. The output will be "Length of the string is: 13", as length() returns the number of characters in the string. 

                          
                      String str = "Hello, world!";
                      System.out.println("Length of the string is: " + str.length());

    • String.substring()

      This Java code snippet initializes a String variable str with the value "Hello, world!". It then prints a substring of str from index 7 to index 12 using the substring() method of the String class. The output will be "Substring from index 7 to 12 is: world", as substring() extracts characters starting at the first specified index up to, but not including, the second specified index. 

                          
                      String str = "Hello, world!";
                      System.out.println("Substring from index 7 to 12 is: " + str.substring(7, 12));

    • String.equals()

      This Java code snippet initializes two String variables str1 with the value "Hello" and str2 with the value "World". It then compares these two strings using the == operator, which checks for reference equality rather than value equality. Since str1 and str2 refer to different objects, the output is false. The == operator in Java checks if both references point to the exact same object, not if their contents are the same. 

                          
                      String str1 = "Hello";
                      String str2 = "World";
                      // Comparing str1 and str2
                      System.out.print(str1 == str2); // output : false

    • toUpperCase() and toLowerCase()

      This Java code snippet initializes a String variable s with the value "Hello World!". It then utilizes two methods from the String class to manipulate the case of the string: - toUpperCase() is called to convert all characters in s to uppercase, resulting in the output "HELLO WORLD!". - toLowerCase() is called to convert all characters in s to lowercase, resulting in the output "hello world!".

      These methods are useful for standardizing the case of strings, especially in scenarios where case-insensitive comparisons are necessary. 
                          
                      String s = "Hello World!";
                      System.out.println(s.toUpperCase()); // Outputs: "HELLO WORLD!"
                      System.out.println(s.toLowerCase()); // Outputs: "hello world!"
