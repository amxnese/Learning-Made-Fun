Conditional Statements in Java

What are Conditional Statements

Conditional statements in Java are used to perform different actions based on different conditions. They allow the code to make decisions and execute certain parts of code only when specific conditions are met. Here are the main types of conditional statements in Java:

  • 1. if statement: Checks a condition and executes a block of code if the condition is true. 
    if (condition) {
  // code to be executed if condition is true
}
  • 2. if-else statement: Checks a condition and executes the first block of code if the condition is true, otherwise executes another block of code. 
    if (condition) {
  // code to be executed if condition is true
} else {
  // code to be executed if condition is false
}
  • 3. else-if ladder: A series of conditions to be checked in sequence. 
    if (condition1) {
  // code to be executed if condition1 is true
} else if (condition2) {
  // code to be executed if condition2 is true
} else {
  // code to be executed if all previous conditions are false
}
  • 4. switch statement: Specifies many alternative blocks of code to be executed. The switch statement is often more efficient than a set of nested if statements when dealing with multiple conditions on the same variable. 
    switch (expression) {
  case value1:
    // code to be executed when expression equals value1
    break;
  case value2:
    // code to be executed when expression equals value2
    break;
  default:
    // code to be executed if expression doesn't match any case
}

When To Use Conditional Statements

Conditional statements are essential in programming for controlling the flow of execution based on specific conditions. Here are some common scenarios when you might use conditional statements:

  • Decision Making: Perform actions based on specific conditions or inputs, such as checking user authorization before accessing features.
  • Handling Different Cases: Execute different blocks of code for various cases or scenarios, like calculating discounts based on the quantity purchased.
  • Input Validation: Ensure that inputs meet certain criteria before proceeding, such as verifying that values are within an expected range.
  • Branching Logic: Direct program execution along different paths based on variable states or user choices, such as selecting operation modes.
  • Error Handling: Manage errors or unusual situations gracefully through conditional checks.

Keywords Related to Conditional Statements in Java

Conditional statements in Java determine which segments of code get executed based on the evaluation of Boolean expressions. Here are essential keywords and concepts related to conditional logic in Java:

  • if - Executes a block of code if the specified condition is true.
  • else - Used to execute an alternate block of code if the condition in the if statement is false.
  • else if - Provides a new condition to test if the previous condition was false.
  • switch - Executes one block of code among many options based on the value of a variable. Often used as an efficient alternative to multiple if statements.
  • case - Used within a switch statement to specify a block of code to be executed.
  • default - Used within a switch statement to specify the default block of code that is executed when no case matches.
  • break - Used to exit a switch case block, preventing the execution of subsequent case blocks.
  • ?: (ternary operator) - A shorthand for if-else, which is used to assign or return a value based on a condition.
  • instanceof - Checks whether an object is an instance of a specific class or implements an interface.

Examples of Using Conditional Statements in Java

  • if statement 
    if (score >= 70) {
  System.out.println("Passed!");
}
  • if-else statement 
    if (age < 18) {
  System.out.println("You are a minor.");
} else {
  System.out.println("You are an adult.");
}
  • else-if ladder 
    if (grade >= 90) {
  System.out.println("Grade A");
} else if (grade >= 80) {
  System.out.println("Grade B");
} else if (grade >= 70) {
  System.out.println("Grade C");
} else {
  System.out.println("Grade F");
}
  • switch statement 
    switch (day) {
  case 1:
    System.out.println("Monday");
    break;
  case 2:
    System.out.println("Tuesday");
    break;
  case 3:
    System.out.println("Wednesday");
    break;
  default:
    System.out.println("Another day");
}
